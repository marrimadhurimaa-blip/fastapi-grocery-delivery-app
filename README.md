Grocery Delivery FastAPI Project:

Description:
This project is a backend API developed using FastAPI. It simulates a grocery delivery system, providing functionality for managing products, handling user carts, and processing orders.

Features:
RESTful GET and POST APIs
Data validation using Pydantic
Full CRUD operations
Cart management and checkout workflow
Search, sorting, and pagination support
Interactive API testing with Swagger UI
Tech Stack
Python
FastAPI
Uvicorn
How to Run

Install dependencies:
pip install fastapi uvicorn

Run the server:
uvicorn main:app --reload

Open the API documentation:
http://127.0.0.1:8000/docs
